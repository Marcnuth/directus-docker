## Quick Start

Install all dependencies and build the image by running:

```bash
bash bin/build
```

Then start the app using any of these methods:

### Using script (use docker run)

```bash
bash bin/run
```

### Using docker-compose
```bash
# Start directus
docker-compose up
```

### Using docker run

[See bin/run](bin/run)
